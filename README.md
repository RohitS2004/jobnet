# What is Jobnet?
- Jobnet is an application on which companies and agents can to hire new talent according to their job requirements. The application mainly focuses on seamlessly apply for a particular job with just a swipe, literally with only one swipe and that's it, your application gets submitted.

 # Why Jobnet?
 - There are already many apps available in the market which you can use in order to apply for jobs, one of the most famous is Linkedin. But with Linkedin what happens is that the process of applying to jobs is becoming more and more complicated and also the jobs that are listed there does not have any true validiation. That's where Jobnet jumps in, it only allows jobs which are validated from our end and are not some sort of scam. One of the biggest thing that differentiate Jobnet from Linkedin is that Linkedin is becoming more like a Professional Social Media platform but Jobnet is solely made for applying for jobs and browsing jobs.
